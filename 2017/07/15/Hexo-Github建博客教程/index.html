创建Github pages

  • 在线创建库并起名yaro97.github.io,在设置中通过Launch automatic page generator生成github page即可
    注意:name要是唯一的话,生成的网址为name.github.io,若不唯一,网址为yaro97.github.com/name.github.io

部署本地Hexo到Github pages

  • 注意部署前一定要安装插件hexo-deployer-git.前面已经安装

  • 修改_config.xml文件

  • 执行命令hexo d完成部署,过程中需要github账号/密码,账号为yaro97.

对B电脑的操作如下：

  1. 安装Git，并配置github账号下的B电脑的.ssh
  2. 安装Node.js

  3. 使用npm指令安装Hexo

  4. 使用Git bash随便选择一个文件夹,执行git clone

至此，两个电脑的hexo环境一致，Hexo博客源文件一致.倘若还有C电脑,重复B电脑的操作即可.

关于日常的改动流程（A，B两台电脑均使用的情况下）

  1. 建议先检查更新git pull，将本地博客源文件更新至最新版本

  2. 然后可以新建或修改博客内容，进行预览等操作

  3. 新建博客后，先同步Hexo源文件，将修改后的源文件同步至github

  4. 然后再执行Hexo的生成文件和部署指令
